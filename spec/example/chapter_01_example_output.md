# 1. 有一个小姐死了

## Characters in this chapter

- 李亮
- 打电话的男人

## Script

打电话的男人: 有一个小姐……死了！她在地上，在房间的地上……她像在睡觉。她很漂亮，但是她……她已经死了！

[打电话的男人说话有点儿快。听电话的是警官李亮。]

[这是在中国的北京，这一天是6月9号，星期五，时间是上午11点15分。三分钟前，李亮警官的大学同学给他打来一个电话说，给他介绍了一个女朋友，李亮打算明天晚上和她见面。]

李亮: 在那儿等着，不要让人进去！

[李亮警官说完，马上跟王明警官一起走出房间，上了汽车，他们要很快到死人的地方去。]
